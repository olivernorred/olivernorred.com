A vector solution

Rob Stenson’s Python library Coldtype inspires me. Its animation tools are extremely powerful, but, as Stenson writes,

I always found it awkward to program animations in JS, since I never found a good way to run a headless browser when I needed to rasterize frames for an animation.

Coldtype isn’t intended for headers or logos on a website; rather, for animations in videos.

Enter the opentype.js library

This library takes a font and a string and returns an SVG Path object I can muck around with in JavaScript.

I mucked around enough that others can now use a script I wrote to define a function for their own animations.
